# IdleonGame

Unity 2D horizontal RPG project skeleton.

## Current scope

- Project structure only.
- Tilemap-oriented map organization.
- Reserved modules for player movement, combat, rope climbing, portals, navigation, and jobs.
- No gameplay implementation yet.

## Asset root

All project-owned assets should live under `Assets/_Project/`.

## Next recommended step

Open this folder with Unity Hub, let Unity generate Library files, then create the first Boot scene and one small Tilemap map scene.
